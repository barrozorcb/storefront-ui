# Odoo template vsf

Template base to start new odoo vsf projects using vsf SDK

## Tech stack

1. Nuxt 3
2. VSF 2
3. SFUI-2

## How to start ?

```sh
2. yarn install
3. cp -f .env.example apps/web/.env
4. export $(cat apps/web/.env | grep -v \#)
5. yarn dev
6. You can access with http://localhost:3000
```
